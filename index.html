Aspxmediumtrusttest 40

Test your .NET 4.0 medium trust environment.

Download this project as a .zip file Download this project as a tar.gz file

Installation instructions:

  1. Download zip file above.
  2. Upload just default.aspx to the web root of your web site via FTP.
  3. Browse to the file location in browser: i.e http://www.yourdomain.com/default.aspx

Usage

This test will allow you to verify whether you have setup correct permissions in your medium trust environment. In addition most shared hosting providers only provide medium trust however some may or may not provide access to System.IO. This script will help you to easily identify this.

Updating

To update to the lastest version simply re-install using instructions above
